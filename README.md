# Procédure d'installation

## Installation projet Back-end
- Cloner le repository `https://github.com/linamandresy/MBDS_Madagascar_Angular_BackEnd_Tsiky_Lina`

- Exécuter la commande suivante pour ajouter les packages nécessaires
```
npm install
```
- Démarrer le projet puis suivre le lien indiqué dans la console
```
npm start
```
## Installation projet Front-end
- Cloner le repository `https://github.com/lenn-tiky02/Angular2022_2023_MBDS_MadagascarFront-main-Tsiky-Lina`

- Exécuter la commande suivante pour ajouter les packages nécessaires
```
npm install
```
- Démarrer le projet puis suivre le lien indiqué dans la console `http://localhost:4200/`
```
ng serve
```

Ensuite pour vous connecter au site vous pouvez utiliser les identifiants suivants:
 - Admin: `admin/admin`
 - Utilisateur simple: `linamandresy/cmcm`
